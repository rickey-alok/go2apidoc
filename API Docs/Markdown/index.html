### H3

##### H5

###### H6

==========

Header 2

    Formatting

    Normal = Normal

    **Bold** = Bold __Bold__ = Bold

    *Italic* = Italic _Italic_ = Italic

    ***Bold & Italics*** = Bold & Italics ___Bold & Italics___ = Bold & Italics

    Subscript example: H~2~O = H2O

    Superscript example: X^2^ = X2

    ==Highlight== = Highlight

    ~~Strikethrough~~ = Strikethrough

    Emojis

    :heart: = ❤️

    :icecream: = 🍦

    :school: = 🏫

    :car: = 🚗

    List

    Ordered list

    Input

    1. Item 1
2. Item 2
   1. Item 2.1
      1. Item 2.1.a
      2. Item 2.1.b

    Output

    1. Item 1
    2. Item 2
    3. Item 2.1
      1. Item 2.1.a
      2. Item 2.1.b

    Unordered list

    Input

    - Item 1
- Item 2
  - second level 1
    - third level
  - second level 2
    - third level

    Output

    • Item 1
    • Item 2
    • second level 1
      • third level
    • second level 2
      • third level
